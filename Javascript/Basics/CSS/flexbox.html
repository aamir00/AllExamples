What is Flexbox?

  • A layout mode to arrange items within a container
  • No floats
  • Rsponsive and mobile friendly
  • Positioning child elements much easier
  • Flex container margin do not collapse with the margin of its content
  • Order of the elements can easily be changed without editing the source html

Flexbox Concept

  • Ability to alter item width and height to best fit in its container
  • No floats
  • Flexbox is direction agnostic
  • Built for small scale layouts while the upcoming "Grid" specification is for large scale
